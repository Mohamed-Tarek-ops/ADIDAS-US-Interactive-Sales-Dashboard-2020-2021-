## 📝 Project Overview

This project is a personal **Data Analysis** initiative focused on analyzing Adidas sales performance across the United States during 2020–2021.

The goal of this dashboard is to transform raw sales data into clear, interactive, and decision-ready insights using Microsoft Excel. It provides a high-level business overview while allowing detailed exploration through dynamic filters.

Data analysis is a critical skill in today’s business environment, enabling organizations to make informed decisions based on real performance indicators. Through this project, I applied analytical thinking, data modeling, and visualization techniques to extract valuable insights.

I am continuously striving to improve and develop my skills in Data Analysis and Business Intelligence, and this project represents a step forward in that journey.

---

## 🎯 Project Objectives

- Analyze overall sales and profitability performance  
- Identify top-performing products and retailers  
- Compare quarterly trends and detect seasonality  
- Evaluate sales performance by region and state  
- Analyze sales channels (In-store, Online, Outlet)  
- Build a fully interactive and user-friendly dashboard

## 📈 Key Insights

- The West region shows strong overall profitability.  
- Sales peak during Q3, indicating seasonal demand patterns.  
- A few major retailers contribute a significant share of total revenue.  
- Online sales represent a growing and impactful sales channel.

## 🚀 How to Use

1. Download the Excel file.  
2. Open it using Microsoft Excel (2016 or later recommended).  
3. Use the slicers to filter by region, state, or quarter.  
4. Explore interactive charts and insights.
